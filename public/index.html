About Vibe Printing

Vibe Printing is a personal content automation project developed by Connor Cress. It generates short-form educational video content for the Compressed Curiosity channel across YouTube Shorts and TikTok.

The application uses AI language models for scriptwriting, text-to-speech for narration, stock footage APIs for visuals, and ffmpeg for video assembly. Content is reviewed before publishing to ensure accuracy and platform compliance.

Vibe Printing integrates with the TikTok Content Posting API to publish approved Shorts directly to TikTok, with full synthetic-media disclosure as required by platform guidelines.

Privacy Policy

Last updated: April 4, 2026

What data we collect

Vibe Printing does not collect personal data from end viewers. When publishing content to TikTok via the Content Posting API, the application accesses only the authenticated user's own account to upload video content.

TikTok API usage

  • We use the TikTok Content Posting API solely to publish video content to the authenticated account.
  • We request only the permissions necessary for content posting (e.g., video.publish, video.upload).
  • We do not access, store, or share any TikTok user data beyond what is required for the publishing flow.
  • OAuth tokens are stored locally and are never transmitted to third parties.

Data storage

All generated content (scripts, audio, video files) is stored locally on the operator's machine. No user data is stored in external databases or cloud services.
