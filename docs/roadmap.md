# Roadmap

## Phase 0: foundation

Deliverables:

- channel thesis
- content lane definitions
- GenSec policy
- provider shortlist
- environment and secrets model

Exit criteria:

- one recommended channel identity
- one approved MVP workflow shape

## Phase 1: local engine

Deliverables:

- prompt templates
- typed output schemas
- topic scoring
- script generation
- scene plan generation
- publish package generation

Exit criteria:

- produce a complete Short package on disk without manual copy/paste

## Phase 2: n8n workflow

Deliverables:

- scheduled workflow
- failure and retry paths
- approval queue
- YouTube upload path
- run logging

Exit criteria:

- one button or schedule produces a held-for-review Short package end to end

## Phase 3: assisted publishing

Deliverables:

- daily batch generation
- review checklist
- publish metadata defaults
- analytics collection

Exit criteria:

- publish `1-2` Shorts daily with operator approval

## Phase 4: guarded autopublish

Deliverables:

- low-risk lane allowlist
- auto-publish thresholds
- dead-letter queue
- spend budgets
- disclosure automation

Exit criteria:

- publish `2-3` Shorts daily with exception-based human intervention

## Phase 5: optimization

Deliverables:

- format experiments
- hook library
- style library
- retention feedback loop
- provider A/B tests

Exit criteria:

- keep daily volume while improving view velocity and completion rate
